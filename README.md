# 물류 관리 프로젝트

## 1. 프로젝트 소개
입고, 출고, 재고 관리를 처리할 수 있는 물류 관리 웹 프로젝트

## 2. 프로젝트 목표
- 물류 흐름(입고 → 재고 → 출고) 구현

## 3. 기술 스택
- Java
- Spring Boot
- MyBatis 또는 JdbcTemplate
- MySQL 또는 PostgreSQL
- HTML / CSS / JavaScript
- Git / GitHub

## 4. 주요 기능
- 상품 관리
- 거래처 관리
- 창고 관리
- 입고 등록 및 조회
- 출고 등록 및 조회
- 재고 조회

## 5. 화면 구성 예정
- 대시보드
- 상품 관리 화면
- 거래처 관리 화면
- 창고 관리 화면
- 입고 관리 화면
- 출고 관리 화면
- 재고 조회 화면

## 6. 진행 계획
